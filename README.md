# Mines and Dungeons

This is a 2D-made text game, which was coded in java using IntelliJ IDEA. This game uses simple command for better experience, with randomly generated map it is everytime new adventure. 

## Characters and Objects:

* P - player
* E - enemy
* B - boss
* W - weapon
* C - consumable
* S - shop
* ch - chest

### ***** - entrance to another room

## Functions:

* Movement: (wsad)
* ChangeSlot: 1/2 (+/ě)
* Attack: f
* DropItem: q
* Interact e
* ESC: esc It's used for opening end closing menus, like help or shop menu.

## Objective:
Your main goal is to fight through the maze. Then find exist.
This maze contains 10 levels, each with many dangers to face.
